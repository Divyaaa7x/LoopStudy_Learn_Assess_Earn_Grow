# LoopStudy - Learn, Assess, Earn & Grow

## 🚀 Project Overview

LoopStudy is a comprehensive learning platform that enables you to **learn new skills**, **assess your knowledge**, **earn rewards**, and **grow your expertise** through an interactive web experience.

[![Live Demo](https://img.shields.io/badge/View%20Live%20App-http-blue?style=flat-square)](https://divyaaa7x.github.io/LoopStudy_Learn_Assess_Earn_Grow/)

## 🌟 Key Features

### 🎓 Learning Portal
- Browse courses and lessons
- Interactive learning materials
- Progress tracking dashboard

### 📝 Assessment System
- Dynamic quizzes and assessments
- Real-time grading
- Detailed performance analytics

### 🏆 Rewards & Achievements
- Point-based reward system
- Achievement badges
- Leaderboards

### 👨‍🏫 Mentorship
- Expert mentor connections
- Booking system for mentorship
- Personalized guidance

### 📱 Responsive Design
- Mobile-friendly interface
- Cross-browser compatibility
- Optimized for all devices

## 🛠️ Tech Stack

### Frontend
- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling and animations
- **JavaScript** - Interactive functionality
- **Vanilla JS** - No framework dependencies

### Backend
- **Java** - Achievement engine
- **Python** - Assessment engine
- **REST APIs** - Core business logic

### Assets
- **Favicon** - Custom favicon.png
- **Icons** - FontAwesome, Bootstrap Icons
